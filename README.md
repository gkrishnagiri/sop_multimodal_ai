# SOP Multimodal AI

Enterprise-grade multimodal AI system for generating detailed SOPs from audio-video recordings.

## Features

- AV Upload
- Speech-to-text
- Speaker diarization
- OCR extraction
- Workflow understanding
- SOP generation
- LangGraph orchestration
- MCP tool integration
- A2A agent communication

## Tech Stack

- FastAPI
- React
- LangGraph
- OpenAI A2A
- MCP
- Whisper
- PaddleOCR
- Docker
- PostgreSQL
